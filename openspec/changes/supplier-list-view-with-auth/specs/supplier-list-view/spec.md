## MODIFIED Requirements

### Requirement: Табличное отображение списка поставщиков
Система SHALL отображать поставщиков на HomePage в виде таблицы вместо карточек.

Столбцы:
1. Название - ссылка на детальную страницу
2. Город
3. Категории - бейджи
4. Мин. сумма заказа
5. Сертификаты - иконка да/нет
6. Заметка - inline редактирование (авторизован) или placeholder

#### Scenario: Отображение списка
- **WHEN** пользователь открывает HomePage
- **THEN** отображается таблица с 6 столбцами

#### Scenario: Фильтрация
- **WHEN** пользователь применяет фильтры
- **THEN** таблица обновляется

#### Scenario: Пагинация
- **WHEN** поставщиков больше одной страницы
- **THEN** отображается пагинация

#### Scenario: Редактирование заметки
- **WHEN** авторизованный пользователь вводит текст в поле заметки
- **THEN** заметка автосохраняется с debounce 500мс

#### Scenario: Неавторизованный пользователь
- **WHEN** неавторизованный пользователь видит таблицу
- **THEN** столбец заметки показывает placeholder Войдите для добавления заметки
