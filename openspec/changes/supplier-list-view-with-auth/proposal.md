## Why

На главной странице (HomePage) поставщики отображаются в виде карточек (SupplierCard), что неэффективно для быстрого просмотра и сравнения ключевых параметров. Табличное представление позволит сразу видеть основные характеристики поставщиков: город, категории, минимальную сумму заказа, наличие сертификатов. Также требуется добавление пользовательских заметок — комментариев к поставщикам, которые видны только авторизованному пользователю. Для этого необходима система авторизации.

## What Changes

- Замена карточного отображения поставщиков на табличное (список) на HomePage
- Добавление столбцов: название, город, категории, мин. сумма заказа, сертификаты, пользовательская заметка
- Реализация простой авторизации по логину и паролю с использованием JWT (Flask-JWT-Extended)
- Новая модель User для хранения учётных данных
- Новая модель SupplierNote для хранения пользовательских заметок (связь user_id + supplier_id)
- API для регистрации, логина и управления заметками
- Frontend: страницы Login/Register, AuthContext, защищённые маршруты

## Capabilities

### New Capabilities

| Capability | Description |
|---|---|
| Авторизация пользователей | Регистрация и вход по логину/паролю с выдачей JWT-токена |
| Пользовательские заметки | Привязка текстовой заметки к поставщику, видимой только автору |
| Табличный список поставщиков | Отображение поставщиков в виде таблицы с ключевыми столбцами |

### Modified Capabilities

| Capability | Change |
|---|---|
| Каталог поставщиков (HomePage) | Карточки заменяются на таблицу; добавляется столбец с пользовательской заметкой |

## Out of Scope

- Роли и права доступа (admin/user)
- Восстановление пароля
- OAuth/социальная авторизация
- Редактирование профиля пользователя
- Email-подтверждение регистрации
