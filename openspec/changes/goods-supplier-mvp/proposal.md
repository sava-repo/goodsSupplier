## Why

Для малого и среднего бизнеса в food-сфере (кафе, рестораны, пекарни, магазины) поиск надёжных поставщиков ингредиентов, готовой продукции и упаковки — рутинная, но критически важная задача. Сейчас информацию приходится собирать вручную из разных источников, что занимает много времени и затрудняет сравнение условий. MVP сервиса позволит централизовать информацию о поставщиках, ускорить поиск и упростить принятие решений.

## What Changes

- Создание веб-сервиса для поиска и сравнения поставщиков продуктов питания
- Реализация каталога поставщиков с фильтрацией по категории товара и региону
- Карточка поставщика с полной информацией: контакты, сайт, минимальный заказ, цены, сертификаты, условия доставки
- Возможность добавления и редактирования информации о поставщиках
- Простой и понятный интерфейс для быстрого поиска и сравнения
- REST API на Flask для управления данными
- React SPA для пользовательского интерфейса
- PostgreSQL для хранения данных

## Capabilities

### New Capabilities
- `supplier-catalog`: Каталог поставщиков — CRUD операции, поиск, фильтрация по категории и региону, карточка поставщика с детальной информацией
- `product-categories`: Категории товаров — управление справочником категорий (ингредиенты, готовая продукция, упаковка и т.д.)
- `supplier-search`: Поиск и фильтрация — поиск по названию, фильтрация по категории, городу/региону, диапазону цен, минимальному объёму заказа
- `supplier-comparison`: Сравнение поставщиков — выбор нескольких поставщиков для параллельного просмотра ключевых параметров

## Impact

- **Новая кодовая база**: Flask-приложение (бэкенд), React-приложение (фронтенд), схема PostgreSQL
- **API**: REST API для управления поставщиками, категориями и поиска
- **Зависимости**: Flask, SQLAlchemy, PostgreSQL, React, Node.js/npm
- **Инфраструктура**: Docker Compose для локальной разработки (Flask + PostgreSQL + React)
